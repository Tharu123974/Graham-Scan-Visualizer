# Graham-Scan-Visualizer
This website is used to visualize the Graham Scan Algorithm.
You can plot random points and you can lock and unlock the canvas to prevent adding of points by misclick.
You can download the trace of the algorithm into a Text File (with .txt extension).
You can choose the suitable theme of your choice from the list of themes given in the website.
The result you can see either as a full animation or step wise process.
